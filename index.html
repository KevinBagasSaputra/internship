      Teori

      Pada hari pertama Praktik Kerja Lapangan (PKL) di APKOMINDO, kami memulai dengan pengenalan dasar tentang Arduino. Kami mendalami lebih dalam tentang konsep dasar Arduino, apa itu Arduino, dan cara kerjanya.

      Bagi saya, itu adalah pengalaman yang membuka mata saat kami menjelajahi dunia mikrokontroler dan pemrograman. Para mentor membimbing kami melalui proses tersebut, memastikan bahwa kami memahami prinsip dan konsep penting Arduino. Pengalaman awal ini menjadi dasar yang kuat untuk proyek dan eksperimen kami di masa mendatang.

      1. Anatomi Arduino

      Meskipun semua papan Arduino berbeda satu sama lain, ada beberapa komponen utama yang dapat ditemukan dihampir semua Arduino. Mari kita lihat gambar dibawah ini:

      1. Mikrokontroler - ini adalah otak Arduino, dan merupakan komponen tempat kita memuat program. Anggap saja ini sebagai komputer kecil, yang dirancang untuk menjalankan sejumlah hal tertentu saja.
      2. Port USB - digunakan untuk menghubungkan papan Arduino Anda ke komputer.
      3. Chip USB ke Serial - USB ke Serial merupakan komponen penting, karena membantu menerjemahkan data yang berasal dari misalnya komputer ke mikrokontroler on-board. Inilah yang memungkinkan untuk memprogram papan Arduino dari komputer Anda.
      4. Pin digital - pin yang menggunakan logika digital (0,1 atau LOW/HIGH). Umumnya digunakan untuk sakelar dan untuk menyalakan/mematikan LED.
      5. Pin analog - pin yang dapat membaca nilai analog dalam resolusi 10 bit (0-1023).
      6. Pin 5V / 3.3V - pin ini digunakan untuk memberi daya pada komponen eksternal.
      7. GND - juga dikenal sebagaigroundBahasa Indonesia:negativeatau hanya-, digunakan untuk melengkapi suatu rangkaian, di mana level listriknya berada pada 0 volt.
      8. VIN - singkatan dari Voltage In, tempat Anda dapat menghubungkan catu daya eksternal.

      2. Konsep Dasar Arduino

      Ada beberapa konsep dasar yang perlu kamu ketahui tentang Arduino:

      1. Microcontroller (MCU) : MCU adalah otak dari sistem Arduino. Ini adalah bagian utama yang mengolah data dan menjalankan instruksi.
      2. Input/Output (I/O) : I/O adalah proses transfer data antara MCU dengan komponen lainnya. Ada dua jenis I/O, yaitu input (masuk) dan output (keluar).
      3. Circuit Board : Circuit board adalah papan sirkuit yang berisi komponen-komponen elektronika, termasuk MCU.
      4. Programming : Programming adalah proses menulis instruksi untuk membuat sistem Arduino bekerja.

      3. Apa itu Arduino

      Arduino adalah sebuah platform mikrokontroler yang digunakan untuk membuat proyek-proyek elektronik seperti robot, sensor, dan lain-lain. Mikrokontroler ini memiliki kemampuan mengolah data digital dan analog, serta dapat dihubungkan dengan berbagai jenis komponen.

      4. Cara Kerja Arduino

      1. User Input : Pengguna memberikan input melalui tombol sensor, atau komponen lainnya.
      2. MCU Mengolah Data : MCU menerima input dan mengolah data tersebut.
      3. Output : Hasil output ditampilkan pada layar LCD, dihubungkan ke perangkat lain, atau digunakan sebagai sinyal.
      4. Looping : Sistem Arduino dapat berulang-ulang melalui proses input-output untuk membuat sistem yang berkelanjutan.
